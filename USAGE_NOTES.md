Utilisation & notes

- Pour connecter un *ou plusieurs appareils*: Ouvrez WhatsApp → Menu ⋮ → Appareils connectés → Scanner le QR depuis la page web fournie par le bot.
- Tapez `.*gmenu*` dans WhatsApp pour obtenir un menu détaillé, propre et professionnel avec toutes les catégories.
- Chaque commande renvoie une image manga aléatoire et une signature `powered by precieux okitakoy`.

Note: Node.js 18+ est recommandé pour le déploiement sur Render.
